# Roadmap — miles-app

# Phase 1 — MVP Foundation

## Goal

Validar monitoramento automatizado de emissões com milhas.

## Deliverables

- estrutura inicial do projeto
- scrapers iniciais
- armazenamento de resultados
- sistema de alertas
- notificações por e-mail

---

# Phase 2 — Product Expansion

## Goal

Aumentar retenção e utilidade do produto.

## Deliverables

- dashboard web
- histórico de preços
- alertas por WhatsApp
- novos programas de milhas
- filtros avançados

---

# Phase 3 — Intelligence Layer

## Goal

Transformar dados acumulados em vantagem estratégica.

## Deliverables

- score de oportunidade
- previsão de preços
- detecção de anomalias
- ranking de melhores emissões
- insights personalizados

---

# Phase 4 — Platform

## Goal

Expandir o miles-app para uma plataforma de inteligência de viagens.

## Deliverables

- APIs públicas
- marketplace
- integrações externas
- automações avançadas
- ecossistema de parceiros
